# Skill: Job Description Analysis

**Description**: How to analyze and extract key information from a job description.

Approach this analysis as an expert technical recruiter reading the JD with a candidate's profile in mind.

## Extraction Targets
- **Language**: Determine if the description is in DE, EN, etc.
- **Technical Skills**: Frameworks, languages, tools, platforms explicitly required or preferred.
- **Soft Skills**: Communication, leadership, collaboration signals.
- **Responsibilities**: Core tasks required by the role.
- **Company Focus**: Are they product-driven, mission-driven, tech-driven?

## Output format
Return a structured output with:
- `keywords`: list of critical terms.
- `priorities`: ranked list of what the company values most.
